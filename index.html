Introduction to CSS Grid

CSS Grid allows you to create gridded layouts with ease purely through CSS. Below we will examine some of the different properties CSS Grid offers and how each affects the content.

For a full description of each property, check out CSS Tricks'A Complete Guide to Grid, and as always, be sure to read the W3C spec for the complete story on every property and how calculations are done.

grid-template-rows and grid-template-columns

grid-template-rows is a property of the grid container that determines the heights of each of the rows in the grid. grid-template-columns determines the number and width of the columns. By default, there is one column of 100% width and rows are given auto values, which lets the content in the row determine its height. You can give any valid height/width units to these properties, as well as fr units, which determine how much of the remaining available free space in the container to use after the other rows and columns have calculated their dimensions.

Note that defining rows or columns makes space for those rows/columns in the grid container, even if there are no items to fill the space.

grid-column-gap, grid-row-gap, and grid-gap

grid-column-gap and grid-row-gap are properties of the grid container that determine the width of the lines between grid items. If you're familiar with other grid frameworks, this is often referred to as the "gutter". Note that, if you used percentage measurements for grid-template-columns, item width is calculated before the gutter width, which can push content outside the container. grid-gap is a shorthand property that combines the other two properties into one.

justify-items and align-items

justify-items is a property of the grid container that determines the alignment of the content inside each item along the row axis. align-items determines the alignment along the column axis. Valid values are start | end | center | stretch

justify-content and align-content

justify-content is a property of the grid container that determines the horizontal alignment of the grid content inside the grid container. align-content determines the vertical alignment of the content inside the container. Valid values are start | end | center | stretch | space-around | space-between | space-evenly

grid-column-start, grid-column-end, grid-row-start, and grid-row-end

These properties of the grid items determine the placement and size of a given item. align-content determines the vertical alignment of the content inside the container. Valid values are start | end | center | stretch | space-around | space-between | space-evenly
